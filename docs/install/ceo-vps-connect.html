Step 1 of 2 — Join the team network

DepthFusion uses Tailscale to create a private, encrypted network between your computer and the team server. You set it up once and it runs silently in the background forever after.

1
Download and install Tailscale
Go to the Tailscale download page and get the version for your computer.
Browser
https://tailscale.com/download
💡
Open the downloaded file and click through the installer — it's a standard next → next → finish installer. No special options needed.
2
Sign in to Tailscale
After installing, Tailscale will ask you to log in. You can use Google, Microsoft, or create a free account with your email.
💡
Mac: Click the Tailscale icon in your menu bar (top-right of screen) → Log in. Your browser will open — follow the prompts.

Windows: Find the Tailscale icon in your system tray (bottom-right, near the clock — may be hidden behind a ^ arrow). Right-click it → Log in. Your browser will open.
📸 After signing in, the icon will turn from grey to a solid colour. That means Tailscale is running.
3
Message Greg to get approved
Tell Greg the email address you used to sign in to Tailscale. He'll approve your device — this takes about 1 minute on his end.
⚠️
You must wait for Greg's approval before the connection will work. Until approved, you're on Tailscale but not yet on the team network. The icon will be solid but you can't reach the server yet.
4
Get the team address from Greg
Once he approves you, Greg will send you an address that starts with 100. — for example, 100.94.12.34. Enter it in the box at the top of this page and continue to Step 2 below.
Step 2 of 2 — Connect Claude Desktop
🍎 Mac
🪟 Windows
1
Open Terminal
Terminal is the text command window on your Mac. You'll use it once to run a single command.
💡
Press ⌘ Command + Space to open Spotlight search. Type Terminal and press Enter. A black (or white) window will open — that's normal.
📸 You'll see a window with a blinking cursor and text like yourname@MacBook ~ %
2
Run the connection command
Click the copy button, then click in the Terminal window and paste with ⌘ Command + V, then press Enter.
Terminal
claude mcp add depthfusion \ http://YOUR_VPS_IP:7301/sse
⚠️
If you see "command not found: claude", Claude Code CLI isn't installed. Skip to the Manual config method below instead.

You should see: Added MCP server depthfusion

3
Restart Claude Desktop
Fully quit and reopen Claude Desktop so it picks up the new connection.
💡
Click the Claude icon in your menu bar (top right of screen) → Quit Claude. Then open Claude from your Applications folder or Dock.
✓
You're connected

📋 Manual method — if the command above didn't work

This method edits a config file directly. You'll use it if the claude command wasn't found.

Step 1. Open Finder. Press ⌘ Command + Shift + G and paste this path:

Finder path
~/Library/Application Support/Claude/

Step 2. Open claude_desktop_config.json with TextEdit. If it doesn't exist, create it.

Step 3. Replace the entire file contents with this (keep your VPS address in the url line):

JSON config
{ "mcpServers": { "depthfusion": { "url": "http://YOUR_VPS_IP:7301/sse" } } }

Step 4. Save the file (⌘S). Quit and restart Claude Desktop.

1
Open PowerShell

Troubleshooting
The Tailscale icon is grey or says "Disconnected"

Tailscale is installed but not connected. Try these in order:

  1. Mac: Click the Tailscale icon in your menu bar → Connect. If you don't see the icon, open Spotlight (⌘ Space) and search for Tailscale.
  2. Windows: Find Tailscale in your system tray (bottom-right near the clock, might be behind a ∧ arrow). Right-click → Connect.
  3. If it asks you to log in again, sign in with the same account you used before.
  4. If the icon turns solid but the connection still fails, your approval may have expired — message Greg to re-approve you.
I don't see a response when I type depthfusion_status

This usually means Claude Desktop hasn't loaded the new MCP server yet.

  1. Fully quit Claude Desktop (don't just close the window — quit from the menu/tray icon).
  2. Reopen it.
  3. Start a brand new chat (don't use an existing conversation).
  4. Type depthfusion_status again.

If still nothing: confirm the VPS address is entered correctly above and re-run the connection command.

"command not found: claude" on Mac

This means Claude Code CLI isn't installed on your machine. That's fine — use the Manual method (the expandable section inside Step 2 above) which edits a config file directly and doesn't need the CLI.

"claude is not recognized" on Windows

Same as Mac — Claude Code CLI isn't installed. Use the Manual method inside Step 2 for Windows. It takes 2 extra steps but works just as well.

Claude Desktop doesn't have a config file yet

This happens on a fresh Claude Desktop install. Create the file using the Manual method:

  • Mac: Go to ~/Library/Application Support/Claude/ and create claude_desktop_config.json.
  • Windows: Go to %APPDATA%\Claude\ and create claude_desktop_config.json.

Paste the JSON config from the Manual method above, save, and restart Claude Desktop.

I get a connection error when testing

Most connection errors are one of these:

  1. Tailscale not running — check the Tailscale icon. If it's grey or says "Disconnected", click it and reconnect.
  2. Not approved yet — message Greg and make sure he's approved your device in Tailscale.
  3. Wrong address — the team address at the top must start with 100. and have no extra spaces.
  4. Server temporarily offline — message Greg to confirm it's running, then wait 5 minutes and try again.

Quick test: if Tailscale is running and connected, you should see your Tailscale icon as a solid colour (not grey).

I need to remove or reset the connection

To remove DepthFusion from Claude Desktop:

Mac Terminal: claude mcp remove depthfusion -s user

Windows PowerShell: claude mcp remove depthfusion -s user

Manual: Open the config file and delete the "depthfusion" block, then restart Claude Desktop.
