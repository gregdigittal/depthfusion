E-31 Cognitive Infrastructure

Structured Evolving Cognition

Eight co-operating subsystems that give DepthFusion genuine cognitive behaviour: scoring, contradiction detection, autonomic consolidation, and more.

🧠

CognitiveScorer

8-component weighted ranking: semantic 25%, lexical 18%, confidence 15%, regime 12%, graph 10%, recency 8%, historical usefulness 7%, workflow 5%.

⚡

ContradictionEngine

Detects contradictions via negation patterns and ≥40% token overlap. Pinned memories always win. Reduces conflicting context by ~40%.

🔄

MemoryConsolidator

Autonomic background loop (dry-run safe). Identifies merge candidates and archive targets. Never mutates state without explicit invocation.

📦

7 Typed MemoryObjects

decision · semantic · operational · procedural · episodic · social · temporal. SQLite WAL projection for efficient concurrent access.

📋

EventLog

Idempotent fcntl-safe JSONL. 9 event types: CREATED, UPDATED, ACCESSED, SCORED, FEEDBACK, MERGED, ARCHIVED, SUPERSEDED, LINKED.

🌐

REST API

FastAPI on 127.0.0.1:7300. Loopback-safe by design. Public mode requires API token or raises ValueError at startup.

🔗

Multi-Agent Working Memory

Shared working-memory substrate across parallel Claude Code sessions. Context bus via file or Supabase backend.

🤖

LLM Decision Extractor

Haiku-backed structured extraction (precision target ≥0.80 vs heuristic ~0.60). Falls back gracefully when backend unavailable.
