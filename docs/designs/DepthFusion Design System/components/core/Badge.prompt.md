**Badge** — small status pill for document classification and source tags.

```jsx
<Badge variant="confidential">confidential</Badge>
<Badge variant="restricted">restricted</Badge>
<Badge variant="source">Document</Badge>
```

- Classification variants: `public` (olive), `internal` (cold teal), `confidential` (amber), `restricted` (ember). `source` is the neutral amber tag used for source types (Document/Web/Email/Code…).
