What it does

Every layer of the agent stack, covered.

DepthFusion is the persistence layer your AI agent team is missing.

🧠
Persistent Semantic Memory
Entities, relationships, and discoveries stored in a local SQLite knowledge graph. Semantic search via HNSW + BM25. Memory outlasts every context window.
⚡
Instant Session Seeding
fabric_seed mode cold-starts any agent with ranked context: recall_relevance × recency_decay × observer_count weighting. Zero ramp-up time.
🕸️
Event Graph Fabric
Every publish, subscribe, and recall is a first-class graph node. Real-time SSE stream. Redis-backed with graceful degradation. Multi-agent, cross-session.
🔍
Agent Provenance Queries
Ask "who knew what, when" across your fleet. /v1/graph/agent trail and /v1/graph/memory observers endpoints. Full audit trail, queryable at any time.
🔒
Secure Team Sharing
Tailscale + Bearer token auth. Loopback-only by default. DEPTHFUSION_API_TAILSCALE=1 exposes to your Tailscale network — nothing else. Redis stays local.
🛠️
MCP + REST + CLI
27 MCP tools for Claude. REST API for any HTTP client. CLI for direct access. Works with Claude Code, custom agents, and anything that speaks MCP.
