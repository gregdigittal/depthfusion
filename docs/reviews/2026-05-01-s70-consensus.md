3. **`extractor_wiring_required` behavioral probe** — `_probe_extractor_wiring_lazy()` actually writes a tmp file via `write_decisions()` and checks for `importance:` in the output. Replaces the prior import-only probe.
4. **Vacuous assertion removed** — `test_old_file_yields_defaults` now asserts `score is not None` and committed contract (`extract_memory_score` always returns canonical defaults, never None).
5. **NaN/Inf clamp tests added** — `test_non_finite_importance_clamps_to_default` and `test_non_finite_salience_clamps_to_default` (3 parametrize cases each, total 6 new tests). `-0.0` added to the existing importance parametrize. Near-zero and near-max fence-posts added to salience parametrize.
6. **Malformed scalar test added** — `test_malformed_scalars_fall_back_to_defaults` (6 parametrize cases) covers `not-a-number`, alpha-string, NaN-as-string, Inf-as-string, -Inf-as-string, empty-string. Asserts no exception, finite floats only, in-range values.
7. **Concurrent torn-write test added** — `test_set_memory_score_concurrent_writes_no_corruption` spawns two threads racing on the same file with disjoint payloads; asserts the final file parses cleanly to a `MemoryScore` whose `(importance, salience)` pair came from exactly one writer (no field mixing).
8. **MAX-of-confidences design comment** — multi-line docstring on `test_aggregate_importance_uses_max_confidence` documenting the deliberate choice and considered alternatives (mean, last, weighted-mean) with rationale.
